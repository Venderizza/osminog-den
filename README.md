# Осьминог — лендинг ресторана морепродуктов

Цель проекта: простой, элегантный одностраничный сайт (лендинг) для ресторана **«Осьминог»** в эстетике, вдохновлённой **ava.rest**: минимализм, «воздух», крупные фотографии, мягкие морские акценты.

## Готово (реализовано)
- Одностраничный лендинг:
  - Hero-секция с крупным фоном
  - «О ресторане» (кратко, 2–3 предложения)
  - «Шеф‑повар» (минимальная информация)
  - Мини‑галерея (6 фото) + лайтбокс
  - Контакты (адрес/телефон/часы)
- Главная CTA-кнопка **«Забронировать стол»**:
  - В hero-секции
  - Повторяется в конце страницы
  - Ведёт на внешний URL (по умолчанию `https://example.com/booking`)
  - Контрастная (золотая) + hover-эффект
- Упрощённый JS без сложной логики:
  - мобильное меню (бургер)
  - лайтбокс галереи
  - подстановка текущего года в футер
- Плавающая кнопка WhatsApp (ссылка-заглушка)

## Удалено по требованиям
- Раздел «Меню» и фильтры
- Раздел «Доставка»
- Форма бронирования и сохранение заявок через Table API

## Страницы / URI
- `GET /index.html` — главная страница-лендинг

## Где менять ссылку на бронирование
В файле `index.html` замените все вхождения:
- `https://example.com/booking`

(Сейчас ссылка стоит в 3 местах: в шапке, в hero и в CTA-блоке в конце.)

## Настройка контактов
В `index.html` обновите:
- телефон: `+7 (999) 000‑00‑00`
- адрес: `Москва, ул. Примерная, 10`
- часы: `Ежедневно 12:00–23:00`
- WhatsApp: `https://wa.me/79990000000`

## Используемые технологии
- HTML5, CSS3, JavaScript (без сборщиков)
- Шрифты: Google Fonts (Inter, Playfair Display)
- Иконки: Font Awesome (CDN)
- Изображения: временно Unsplash (замените на ваши)

## Хранилище данных / API
- В текущей версии **не используется**.
- RESTful Table API не подключён, так как форма бронирования удалена.
